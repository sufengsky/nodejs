# 使用node读取excel数据

## 1安装所需要的组件
    npm install node-xlsx --save
















```js
    let arr = ['abc', 'def', 'cfd']
    //遍历一个数组的下标
    for (let i in arr) {
        console.log(i)
    }
    //遍历一个数据中的每个元素值
    for (let o of arr) {
        if (o == 'def') break
        console.log(o)
    }
```
